# Usage with the Elastic CI Stack for AWS

There is an official [Terraform module](https://registry.terraform.io/modules/buildkite/elastic-ci-stack-for-aws/buildkite/latest) for deploying Buildkite agents to AWS. There is also a [CloudFormation](https://github.com/buildkite/elastic-ci-stack-for-aws)
stack available.
